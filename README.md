# Daiwahome Copse

## How do I install these formulae?

`brew install daiwahome/copse/<formula>`

Or `brew tap daiwahome/copse` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "daiwahome/copse"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
